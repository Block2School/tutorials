Declare an integer variable named myNumber, assign it a value of 42, and print the value to the console.

Empty code :

```cpp
#include <iostream>

int main() {
    // Declare the variable and assign a value here

    // Fill in the blank here to print the value

    return 0;
}
```

Expected answer: 
```cpp
#include <iostream>

int main() {
    // Declare the variable and assign a value here
    int myNumber = 42;

    // Fill in the blank here to print the value

}
```
