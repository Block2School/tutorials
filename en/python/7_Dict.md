# Dicts

## Définition

On définit un dictionnaire comme ceci:  

```python
myDict = {
    "key": "element",
    "key2": 1,
} # Définition avec des accolades avec une clé et un élément.

myDict.get("key") # Retourne la valeur "element"
myDict["key"] # Retourne également la valeur "element"
```
